## 使用方法
根据你的云崽版本下载对应版本插件，V2放进/lib/example文件夹，V3放进plugins/example文件夹
如何**下载单个插件**：点击"raw"（gitee中是“原始数据”）查看该插件源文件，将网页另存问即可；也可以点击copy代码，粘贴至本地.js文件中
![下载单个插件](./resources/如何下载单个插件.png)
